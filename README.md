# ✈️ AirFreightPredict

항공 화물 가격 예측 머신러닝 프로젝트입니다.

---

## 📌 프로젝트 소개

머신러닝 모델을 활용하여 항공 화물 가격 데이터를 분석하고 예측하는 프로젝트입니다.

---

## 🛠️ Tech Stack

| 분야 | 기술 |
|---|---|
| Language | Python |
| Library | Pandas, NumPy |
| ML | Scikit-learn |
| Visualization | Matplotlib |

---

## ✨ 주요 기능

- 데이터 전처리
- 회귀 모델 학습
- 성능 평가
- 데이터 시각화

---

## 🚀 실행 방법

```bash
pip install -r requirements.txt
python main.py
```
